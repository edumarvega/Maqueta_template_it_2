Qué hacemos

Siguiendo los standares de desarrollo de software, metodologías y patrones de diseño, desarrollamos aplicaciones web, sitios web, y pruebas de usabilidad UX.



Desarrollo de aplicaciones web,utilizando metodologías de desarrollo,esto es, análisis, diseño, programación, testing e implementación.


Desarrollo de sitios web con diseño responsivo, esto es, sitios que se adaptan a distintos dipositivos como celulares, tablet o pc.


Evaluación de usabilidad UX de sitios web, este test permitirá al cliente hacer ajustes en su sitio web para hacer una experiencia positiva del usuario al navegar el sitio web.
